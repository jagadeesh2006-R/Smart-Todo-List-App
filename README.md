# Smart-Todo-List-App
A Streamlit-based To-Do List Application with SQLite database, task priorities, due dates, and analytics dashboard.

# Smart To-Do List App

A task management application built using Streamlit and SQLite.

## Features

- Add Tasks
- Delete Tasks
- Mark Tasks as Completed
- Due Date Tracking
- Priority Management
- Dashboard Metrics
- Interactive Analytics

## Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- Plotly

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
